## Guía de Instalación

easy-docs es una aplicación de Next.js generada con Create Fumadocs. Sigue estos pasos para comenzar:

1. Clona el repositorio:

    ```bash
    git clone https://github.com/DarkInventor/easy-docs.git
    cd easy-docs
    ```

2. Instala las dependencias:

    ```bash
    pnpm install
    ```

3. Ejecuta el servidor de desarrollo:

    ```bash
    pnpm dev
    ```

4. Abre [http://localhost:3000](http://localhost:3000) con tu navegador para ver el resultado.

## Uso

Después de iniciar el servidor de desarrollo, puedes:

-   Editar el contenido en el directorio `pages` para actualizar tu documentación.
-   Modificar los estilos en `styles/globals.css` para personalizar la apariencia.
-   Agregar nuevas páginas creando nuevos archivos `.mdx` en el directorio `pages`.
